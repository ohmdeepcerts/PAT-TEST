Handwriting Scan (AI-powered)

Engineers can photograph a handwritten test sheet on the New PAT Test page and have Asset ID + Description rows filled in automatically. Add a free Gemini API key below for the best accuracy. If left blank, scans automatically fall back to a free public OCR service (no signup) that is far less accurate on handwriting — Gemini is strongly recommended.

Free, no credit card. Get one in under a minute at aistudio.google.com/apikey — sign in with any Google account, click "Create API key", paste it here.
Without a Gemini key
Scans still work using OCR.space's free public demo endpoint, but plain OCR struggles with handwriting — always review results before adding rows either way.

The admin email (set in Company tab) sees Settings + Audit Log. All other registered accounts are Engineers.
